# Patrycja-Oledzka---projekt-Wstep-do-analizy-danych-w-SI
Analiza temperatury w Warszawie 1993-2022 - projekt zaliczeniowy


## Opis projektu
Projekt zaliczeniowy z przedmiotu "Wstęp do Analizy Danych w SI". 
Analiza 30-letniego szeregu czasowego średniej miesięcznej temperatury w Warszawie.

## Cele projektu
1. Analiza trendu klimatycznego w Warszawie
2. Badanie sezonowości temperatury
3. Budowa modelu prognozującego temperaturę
4. Ocena dokładności modelu

Dane do pobrania do projektu (10954 wierszy) na Kaggle:
[https://www.kaggle.com/datasets/mateuszk013/warsaw-daily-weather](https://www.kaggle.com/datasets/mateuszk013/warsaw-daily-weather)
